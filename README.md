# hackaugie-demo

A simple Express.js Task API with in-memory CRUD operations.

## Endpoints

| Method | Path           | Description  |
|--------|----------------|--------------|
| GET    | /api/tasks     | List all     |
| GET    | /api/tasks/:id | Get one      |
| POST   | /api/tasks     | Create       |
| PUT    | /api/tasks/:id | Replace      |
| DELETE | /api/tasks/:id | Delete       |

## Usage

```bash
npm install
npm start
```
